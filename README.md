# 📅 Randevu Yönetim Sistemi

Randevu Spring, kullanıcıların randevu oluşturmasını ve yönetmesini sağlayan bir backend uygulamasıdır.

## 🚀 Features

- 📆 Randevu oluşturma
- ✏️ Randevu güncelleme
- ❌ Randevu silme
- 👤 Kullanıcı bazlı işlemler
- 🔗 RESTful API endpoints

## 🧱 Architecture

- Spring Boot tabanlı backend
- Katmanlı mimari (Controller - Service - Repository)

## 🛠️ Technologies Used

- Java
- Spring Boot
- REST API
- SQL
